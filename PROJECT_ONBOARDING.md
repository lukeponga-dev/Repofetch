# Repository Management Project

## Project Overview

This is **RepoFetch**, a Node.js application for fetching and managing GitHub repositories with AI project focus. The project provides multiple scripts for different repository querying scenarios and supports both console and JSON output formats.

## Key Features

- **AI Project Filtering**: Automatically finds repositories with 'ai' in the name
- **GitHub App Integration**: Secure, high-performance API access
- **Multiple Output Formats**: Console display or structured JSON
- **Auto-save Functionality**: JSON exports saved with timestamps
- **CLI Ready**: Perfect for automation and scripts

## Available Scripts

1. **fetchRepos.js** - AI-focused repository fetcher
2. **get_all_repos.js** - Complete repository inventory
3. **repos_to_json.js** - Advanced JSON export with multiple formats
